1. Compilation command
    make clean
    make

2. Execution Step
    java assignmentP3 <InputFileName> 
    Example: java assignmentP3 /home/Username/Documents/badInput1.txt

3. Output 
   Output will be returned in the second argument given at compile time or if no second argument, than output will be in output.txt file.
   Example: java assignmentP3 badInput1.txt outcome.txt
            java assignmentP3 badInput1.txt
